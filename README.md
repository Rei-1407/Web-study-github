# 🐙 Học GitHub cho dự án Unreal Engine

Website học &amp; luyện tập giúp thành thạo **Git/GitHub** trong bối cảnh làm game với Unreal Engine — học qua bài giảng, quiz và luyện phản xạ gõ lệnh thật.

🔗 **Bản demo:** https://rei-1407.github.io/Web-study-github/

---

## ✨ Tính năng

- **Bài học có lộ trình** với điều hướng trước/sau, callout, bảng, code tô màu và nút sao chép.
- **Quiz tương tác** sau các mục lý thuyết — phản hồi tức thì kèm giải thích.
- **Thử thách gõ lệnh git** — gõ lệnh thật, hệ thống chấm đúng/sai và gợi ý, giúp xây phản xạ.
- **Bảng tra lệnh** gộp theo nhóm, có ô tìm kiếm nhanh.
- **Checklist "tốt nghiệp"** tương tác.
- **Tạo động lực:** thanh tiến độ, điểm XP, cấp độ, huy hiệu, hiệu ứng chúc mừng.
- **Giao diện sáng/tối**, tập trung vào nội dung. Tiến độ lưu ngay trên trình duyệt.

## 🛠 Công nghệ

Web tĩnh thuần: HTML + CSS + JavaScript (không framework, không bước build).

## 🚀 Chạy thử tại máy

Chỉ cần một static server bất kỳ. Dự án kèm sẵn một server nhỏ không phụ thuộc:

```bash
node tools/serve.mjs 4173
# mở http://localhost:4173
```
